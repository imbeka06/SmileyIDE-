# SmileyIDE

SmileyIDE is a semi-offline web-based Integrated Development Environment (IDE) that allows users to code in multiple programming languages, manage files, and execute code seamlessly. It is designed to work both online and offline, providing a robust and user-friendly coding experience.

## Features
- **Multi-language Support**: Write and execute code in JavaScript, Python, Java, C, C++, and HTML/CSS/JS.
- **File Management**: Create, open, save, and delete files directly within the IDE.
- **Code Execution**: Run your code in a secure backend environment.
- **Offline Functionality**: Work offline with cached resources and local file storage.
- **AI Assistant**: Get code explanations, debugging assistance, and optimization suggestions.
- **Customizable Editor**: Choose your preferred language and theme.

## Getting Started

### Prerequisites
- Node.js and npm installed on your system.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/imbeka06/SmileyIDE-.git
   ```
2. Navigate to the project directory:
   ```bash
   cd SmileyIDE
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application
1. Start the backend server:
   ```bash
   node server/index.js
   ```
2. Open `index.html` in your browser to start using the IDE.

### Testing
Run the unit tests:
```bash
npm test
```
